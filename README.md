# 주식회사 클러시 FrontendEnd 과제

<br />

## 브런치 구조

- `main` : 메인 브랜치
- `feature/todo-app` : Todo 애플리케이션 개발을 위한 브랜치
- `feature/calender-app` : Calender 애플리케이션 개발을 위한 브랜치

<br />

## 브랜치 접근 방법

**1. 프로젝트 클론**

```bash
git clone [레포지토리 주소]
cd [프로젝트 폴더]
```

<br />

**2. 브랜치 목록 확인**

```bash
git branch -a
```

**3. 특정 브랜치로 전환 및 접근**

- Todo 애플리케이션

`git checkout feature/todo-app`

- Calendar 애플리케이션

`git checkout feature/calendar-app`
